Acceso al Sistema CliniSalud

Para acceder al Sistema CliniSalud, deberá abrir el navegador Mozilla Firefox, pues solo desde este navegador podrá manejar la aplicación.

Los pasos a seguir para acceder a la aplicación son los siguientes:

  • Ingresar a la dirección: http://clinisaludca.no-ip.org/misalu/app/, mientras se esté accediendo por conexión a internet, y con la dirección: http://192.168.0.253/misalu/app/, mientras se esté accediendo localmente. Ingresando a esta dirección en el navegador, se accederá a la pantalla de Inicio del Sistema. (ver figura 1).

  • Realizar el registro introduciendo nombre de usuario y contraseña.

Figura 1

En la figura 1. se tiene la Pantalla de inicio. Esta consta de:

1. URL: Dirección IP en la que se encuentra ubicada el sistema. En este caso http://clinisaludca.no-ip.org/misalu/app/

2. Nombre de usuario: Nombre que fue asignado por el administrador para ser identificado el usuario.

3. Contraseña: Palabra clave seleccionada por el usuario para el acceso al sistema.

4. Entrar: Envía los datos de usuario y contraseña a la aplicación, los verifica y si son consistentes el usuario accederá a la aplicación sin mayor complicación.

Si deja algún campo vacío sale una ventana emergente con un mensaje indicando que todos los campos son obligatorios. (ver figura 2)

Figura 2

En el caso de haber un error en el nombre de usuario o contraseña saldrá una ventana emergente indicando que hay un error en alguno de los dos datos. (Ver figura 3)


Figura 3
