Registrar una Orden


Se debe acceder al módulo Clientes luego seleccionar Reg Orden. (ver figura 1)

Figura 1

Luego se muestra un cuadro de texto donde se debe colocar el número de Cédula del cliente y se selecciona el servicio. Las órdenes que pueden registrarse son:

  • CARTA AMBULATORIA

  • CARTA DE COMPROMISO

  • CIRUGIA AMBULATORIA

  • CLAVE DE EMERGENCIA

  • EMERGENCIA

  • HOSPITALIZACIÓN AMBULATORIA

  • MATERNIDAD

  • ORDEN DE ATENCIÓN

  • REEMBOLSO - AMBULATORIO

  • REEMBOLSO - MEDICO (HCM)

  • VICIOS DE REFRACCIÓN

Figura 2

Si el usuario es un Coordinador o un usuario con permisología para realizar donativos, al seleccionar Reg Orden, tendrá la opción de escoger el tipo de servicio y tipo de donativo (ver figura 3). Las selecciones de Donativo posibles son:

  • Sin Donativo: La orden se cargará a la cobertura del cliente.

  • Donativo por Responsabilidad Social: La orden no se cargará a la cobertura del cliente.

  • Donativo VIP: Exclusivamente para órdenes autorizadas por el Presidente de CliniSalud.

Figura 3

Al hacer Clic sobre alguna de los tipos de órdenes antes mencionados, puede ver un ejemplo de como realizar el mismo
